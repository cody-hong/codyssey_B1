# 크런치랩 영상 제작 기록

## 시작 기준

- 브랜드: 크런치랩
- 최종 길이 기준: 전체 60초 이내, 씬당 10초 이내
- 영상 흐름: 씬 1 -> 씬 2 -> 씬 3 -> 씬 4 -> 씬 5
- 우선 제작 순서: 씬 1부터 시작

## 현재 확보된 이미지 기준

- 씬 1: 피곤한 한국 직장인 중심 이미지
- 씬 2: 치킨 박스 오픈과 표정 전환 이미지
- 씬 3: 닭다리 히어로 컷 이미지
- 씬 4: 닭다리 한입 베어 무는 이미지
- 씬 5: 브랜드 엔드카드 이미지

## 오디오 메모

- `크런치랩 로고송`은 씬 1부터 전체 배경으로 사용하고, 마지막에 `크런치랩` 나레이션을 얹는다.
- 로고송의 프롬프트는 `Gemini`에서 정리한 뒤 `Gemini 음악 생성 Pro`로 완성했다.

## 도구 선택 메모

- 처음에는 `Kling AI`와 비교하려 했지만, 구독 중인 `Gemini Pro`의 텍스트/이미지/영상 결과가 기대 이상이라 본편 도구로 정했다.
- 음악도 `Gemini` 안에서 프롬프트를 만든 뒤 `Gemini 음악 생성 Pro`로 이어가니 결과가 안정적이어서 채택했다.
- `Kling AI`는 한도 때문에 보너스 과제 1컷에만 사용했다.

## 보너스 과제 메모

- 보너스 씬은 `Kling AI`로 치킨 공중부양 장면을 생성했다.

## 씬 2 영상 메이킹 프롬프트

### 최종 방향

- 기준 이미지: 씬 2 시작 이미지와 씬 2 종료 이미지를 함께 사용한다.
- 목표 동작: 사람이 보이는 화면에서 시작해 치킨 박스 쪽으로 자연스럽게 줌인하고, 껍질이 벗겨지지 않은 닭다리 1개가 올라와 천천히 회전한 뒤 다시 내려오며 전체 샷으로 복귀한다.
- 주의점: 좌우 반전, 거울 효과, 과한 회전, 급격한 컷 전환, 과도한 모션 블러를 피한다.

### 한국어 프롬프트 초안

```text
제공된 참고 이미지를 기본 구도로 사용하세요. 같은 젊은 한국 직장인, 같은 짙은 회색 오피스 캐주얼 셔츠, 같은 늦은 밤 책상, 같은 블랙과 오렌지 계열의 광고 분위기를 유지하세요.

처음에는 인물과 치킨 박스가 함께 보이는 중간 와이드 샷으로 시작합니다. 그다음 카메라가 부드럽게 치킨 박스 쪽으로 줌인해서 치킨이 화면의 중심이 되게 해주세요. 박스 안에서 껍질이 벗겨지지 않은 황금빛 후라이드 닭다리 하나가 천천히 올라오고, 공중에서 한 번 부드럽게 회전해서 바삭한 표면과 튀김옷의 질감이 잘 보이게 합니다. 회전이 끝나면 닭다리가 천천히 다시 박스 안으로 내려갑니다. 마지막에는 다시 인물이 보이는 전체 구도로 자연스럽게 돌아오게 해주세요.

동작은 자연스럽고 현실적으로, 부드럽게 진행하세요. 컷 전환 없이 하나의 연속 장면으로 만들고, 좌우 반전이나 거울 효과는 사용하지 마세요. 과한 회전, 빠른 모션 블러, 급격한 카메라 움직임도 피하세요. 프리미엄 치킨 광고 느낌, 사실적인 김, 바삭한 질감, 부드러운 배경 흐림, 세련된 조명, 16:9 비율로 유지하세요.

금지 요소:
추가 음식 없음, 버거 없음, 샌드위치 없음, 감자튀김 없음, 추가 사이드 메뉴 없음, 텍스트 없음, 워터마크 없음, 만화 같은 움직임 없음, 새로운 인물 없음.
```

## 영상 제작 원칙

- 이미지에서 이미 정해진 구도와 캐릭터를 최대한 유지한다.
- 씬 1은 분위기 시작점, 씬 2는 전환, 씬 3은 히어로 샷, 씬 4는 한입 베어 무는 만족감, 씬 5는 마무리 엔드카드로 둔다.
- 영상은 이미지에 모션만 얹는 방향으로 가되, 필요하면 초점 이동과 카메라 무빙만 최소한으로 사용한다.
- 씬마다 기록할 항목은 `사용 이미지`, `사용 프롬프트`, `사용 도구`, `선정 이유`, `모션 방향`, `결과 평가`다.
