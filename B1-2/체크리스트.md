# B1-2 작업 및 제출 체크리스트

## 사용 방법

- 앞부분은 작업 진행 체크리스트다.
- 뒷부분은 제출 직전 점검 체크리스트다.
- 영상 길이 조건은 원문 충돌이 있으므로 `확인 필요`로 관리한다.

## 1. 브랜드 결정

- [x] 브랜드명 정하기
- [x] 가상 브랜드인지 실제 브랜드인지 정하기
- [x] 타겟 정의하기
- [x] 톤앤매너 정의하기
- [x] USP 정의하기
- [x] 광고 목적 정하기
- [x] 핵심 메시지 1문장 정하기

## 2. 광고 구조 초안

- [x] 우선 목표 길이를 정하기
- [x] 영상 길이 조건을 `전체 60초 이내 / 씬당 10초 이내`로 정리하기
- [x] 전체 씬 수 정하기
- [x] 각 씬의 역할 한 줄씩 정리하기
- [x] 마지막 3~5초 브랜드 인지 장치 정하기

## 3. 스토리보드 작성

- [x] 브랜드 아이덴티티 섹션 작성
- [x] 캠페인 목표 작성
- [x] 핵심 메시지 작성
- [x] 사용 도구 목록 초안 작성
- [x] 씬 1 필수 필드 채우기
- [x] 씬 2 필수 필드 채우기
- [x] 씬 3 필수 필드 채우기
- [x] 씬 4 필수 필드 채우기
- [x] 필요하면 씬 5 필수 필드 채우기

## 4. 씬별 필수 필드

각 씬마다 아래 항목이 들어가야 한다.

- [x] 씬 번호
- [x] 씬 길이(초)
- [x] 목표 메시지 한 문장
- [x] 화면 구성
- [x] 내레이션 또는 화면 카피
- [x] 사용 도구와 사용 목적
- [x] 입력 프롬프트 원문
- [x] 출력 결과 요약 한 줄
- [x] 생성 결과 파일명 또는 링크

## 5. 도구 선택

- [x] 이미지 생성 AI 1개 이상 정하기
- [x] 비디오 생성 또는 변환 AI 1개 이상 정하기
- [x] 오디오 생성 AI 1개 이상 정하기
- [x] 이미지 대체 도구 1개 이상 정하기
- [x] 비디오 대체 도구 1개 이상 정하기
- [x] 오디오 대체 도구 1개 이상 정하기
- [x] 각 도구를 왜 선택했는지 한 줄씩 적기
- [x] 텍스트 프롬프트 생성 도구를 `Gemini Pro`로 정하기
- [x] 이미지 생성 도구를 `Gemini Pro`로 정하기
- [x] 비디오 생성 도구를 `Gemini Pro`로 정하기
- [x] 노래 생성 흐름을 `Gemini` 프롬프트 -> `Gemini 음악 생성 Pro`로 정하기
- [x] 보너스 과제를 `Kling AI` 치킨 공중부양 씬으로 정하기
- [x] 이미지 프롬프트는 1개 템플릿으로 통일하기
- [x] 인물 캐릭터를 젊은 한국 직장인으로 고정하기

## 6. 프롬프트 개선 기록

- [x] 수정 전 프롬프트 1개 남기기
- [x] 수정 전 결과의 문제점 적기
- [x] 수정 후 프롬프트 적기
- [x] 무엇을 왜 바꿨는지 적기
- [x] 수정 후 결과가 어떻게 달라졌는지 적기

## 7. 일관성 및 제작 전략

- [x] 스타일 일관성 유지 방법 정하기
- [x] 캐릭터 일관성 유지 방법 정하기
- [x] 해상도와 비율 기준 정하기
- [x] 크레딧 부족 시 축소 전략 정하기
- [x] 대기열 또는 플랜 제한 시 대체 전략 정하기

## 8. 영상 편집

- [ ] AI 생성 시각 요소 준비하기
- [x] AI 생성 청각 요소 준비하기
- [x] 씬 2 영상 메이킹 프롬프트 정리하기
- [x] 씬 5 로고송 사용 계획 정리하기
- [ ] 컷 편집하기
- [ ] 자막 또는 화면 카피 넣기
- [ ] 마지막 브랜드 인지 장치 넣기
- [ ] 톤앤매너가 크게 흔들리지 않는지 점검하기

## 9. 제약사항 점검

- [ ] 직접 촬영 영상 사용하지 않기
- [ ] 유료 스톡 영상 사용하지 않기
- [ ] 시각 소스가 생성형 AI 결과물인지 확인하기
- [ ] 청각 소스가 생성형 AI 결과물인지 확인하기
- [ ] 핵심 비주얼과 오디오가 AI 생성 결과물 중심인지 확인하기
- [ ] 편집 도구 사용 범위를 통합 편집 수준으로 유지하기
- [ ] 저작권 또는 윤리 이슈가 큰 소재 피하기

## 10. 설명 가능성 점검

- [ ] 왜 이 브랜드를 골랐는지 설명할 수 있는가
- [ ] 왜 이 도구를 골랐는지 설명할 수 있는가
- [ ] 프롬프트를 어떻게 수정했는지 설명할 수 있는가
- [ ] 여러 결과물을 어떻게 하나의 광고로 묶었는지 설명할 수 있는가
- [ ] 필요하면 보너스 과제 선택 이유를 설명할 수 있는가

## 11. 제출 직전 점검

- [ ] 스토리보드 문서가 실제로 있는가
- [ ] 스토리보드 문서를 PDF로 정리했는가
- [ ] 광고 영상 MP4 파일이 실제로 있는가
- [ ] 파일명 규칙이 일관적인가
- [ ] 씬별 필수 항목 누락이 없는가
- [ ] 최소 1개 씬의 프롬프트 수정 전/후 기록이 있는가
- [ ] 최종 영상에 AI 생성 시각 요소가 들어 있는가
- [ ] 최종 영상에 AI 생성 청각 요소가 들어 있는가
- [ ] 영상이 단순 컷 나열이 아니라 메시지 구조를 갖는가
- [ ] 마지막 3~5초에 브랜드 인지 장치가 들어 있는가
- [ ] 영상 길이 기준 해석을 스스로 설명할 수 있는가
- [x] 씬 4를 한입 베어 먹는 장면으로, 씬 5를 엔드카드로 재구성했는가
- [x] 실제 사용 도구를 `Gemini Pro` / `Gemini 음악 생성 Pro` / `Kling AI` 기준으로 기록했는가
