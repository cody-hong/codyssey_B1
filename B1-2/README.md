# B1-2 광고 미션

## 개요

이 폴더는 "AI로 글·그림·영상 만들어 나만의 광고 만들기" 미션 작업 공간이다.

핵심 목표는 브랜드 1개를 정하고, 스토리보드와 AI 기반 광고 영상을 만들어 제출 가능한 패키지로 정리하는 것이다.

## 현재 포함 파일

- `00. 미션 원문.md`
  - 사용자가 제공한 미션 안내문 원문 정리본
- `handoff.md`
  - 현재 상태와 다음 작업 메모
- `체크리스트.md`
  - 작업 진행과 제출 직전 점검을 함께 관리하는 체크리스트
- `01. 스토리보드_크런치랩_브랜드광고.md`
  - 크런치랩 기준 스토리보드 초안
- `02. 광고영상_크런치랩_산출정보.md`
  - 크런치랩 기준 영상 산출 정보
- `03. 영상 제작 기록.md`
  - 씬별 영상 제작 결정과 결과 기록

- 실제 이미지/비디오/오디오 생성 단계로 넘어가기
