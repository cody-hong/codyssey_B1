# B1-3 노코드 자동화 미션

## 개요

이 폴더는 "코딩 없이 마우스 클릭만으로 일이 자동으로 돌아가게 만들기" 미션 작업 공간이다.

핵심 목표는 동일한 자동화 워크플로우를 2개 이상의 노코드 자동화 도구로 직접 구현하고 비교한 뒤, 자유 주제로 실제 동작하는 자동화 파이프라인을 설계·구현하는 것이다.

## 현재 포함 파일

- `00. 미션 원문.md`
  - 사용자가 제공한 미션 안내문 원문 정리본
- `01. 제작 Spec.md`
  - 이번 미션 산출물 제작 기준
- `02. 프로젝트1_자동화도구_비교구현_보고서.md`
  - 2개 이상 자동화 도구 비교 구현 보고서 템플릿
- `03. 프로젝트2_자유주제_자동화_설계문서.md`
  - 자유 주제 자동화 설계 및 구현 문서 템플릿
- `handoff.md`
  - 현재 상태와 다음 작업 메모
- `체크리스트.md`
  - 구현, 캡처, 보고서, 제출 직전 점검 체크리스트

## 추천 진행 순서

1. 프로젝트 1 공통 워크플로우 확정
2. Make로 프로젝트 1 구현 및 캡처
3. Zapier 또는 다른 도구로 같은 구조 구현 및 캡처
4. 비교 분석 보고서 작성
5. 프로젝트 2 자유 주제 확정
6. 선택 도구 1개로 자동 실행 워크플로우 구현
7. 실행 결과와 민감정보 마스킹 확인

## 현재 기본 가정

- 프로젝트 1 도구 후보: Make, Zapier
- 프로젝트 1 워크플로우 후보: Google Forms 또는 Webhook 입력 -> 조건 분기 -> Google Sheets 기록 -> Slack 또는 Discord 알림
- 프로젝트 2 주제: 확인 필요

위 항목은 실제 계정 접근성, 무료 플랜 제한, 사용 가능한 협업 도구에 따라 바꿀 수 있다.
