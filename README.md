# Codyssey B Workspace

## 개요

이 작업공간은 Codyssey B 계열 미션을 폴더별로 분리해 관리하기 위한 루트 작업공간이다.

현재 새로 진행하는 작업은 `B1-3`이며, 기존 B1-1 산출물은 `B1-1`에 보관했다.

GitHub에서도 같은 구조로 보이도록 각 미션의 산출물과 보조 자료는 해당 폴더 안에 모아 관리한다.

## 폴더 구조

- `B1-1/`
  - 이전에 완료한 B1-1 미션 산출물과 보조 자료
- `B1-2/`
  - "AI로 글·그림·영상 만들어 나만의 광고 만들기" 미션 작업 공간
- `B1-3/`
  - 현재 진행 중인 "코딩 없이 마우스 클릭만으로 일이 자동으로 돌아가게 만들기" 미션 작업 공간

## 작업 원칙

- 미션별 산출물은 각 폴더 안에서 관리한다.
- 공통 운영 규칙은 루트 `AGENTS.md`를 먼저 확인한다.
- 현재 세션의 이어가기 기준 문서는 루트 `handoff.md`와 각 미션 폴더의 `handoff.md`다.
